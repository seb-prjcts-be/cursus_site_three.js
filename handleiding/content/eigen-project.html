De live editors zijn fijn om te leren, maar echt werk maak je in een eigen bestand. Deze pagina geeft je de complete startcode en legt uit hoe je een three.js-project op je eigen computer opzet.

Kernpunten
  • Eén index.html met een importmap is genoeg - geen npm, geen build-tools
  • De importmap vertelt de browser waar import 'three' vandaan moet komen
  • Je hebt een lokale server nodig (file:// werkt niet met modules)
  • De startcode hieronder bevat het volledige ritueel, inclusief resize en OrbitControls

De startcode

Kopieer dit als index.html in een eigen map. Dit is de boilerplate die alles uit de cursus combineert - leeg canvas, klaar om te bouwen:

<!DOCTYPE html>
<html lang="nl">
<head>
  <meta charset="utf-8">
  <title>Mijn three.js sketch</title>
  <style> body { margin: 0; overflow: hidden; } </style>
  <script type="importmap">
  {
    "imports": {
      "three": "https://cdn.jsdelivr.net/npm/three@0.184.0/build/three.module.js",
      "three/addons/": "https://cdn.jsdelivr.net/npm/three@0.184.0/examples/jsm/"
    }
  }
  </script>
</head>
<body>
<script type="module">
import * as THREE from 'three';
import { OrbitControls } from 'three/addons/controls/OrbitControls.js';

// --- het ritueel ---
const scene = new THREE.Scene();
scene.background = new THREE.Color(0x141414);

const camera = new THREE.PerspectiveCamera(60, innerWidth / innerHeight, 0.1, 1000);
camera.position.set(3, 2, 5);

const renderer = new THREE.WebGLRenderer({ antialias: true });
renderer.setSize(innerWidth, innerHeight);
document.body.appendChild(renderer.domElement);

const controls = new OrbitControls(camera, renderer.domElement);
controls.enableDamping = true;

// --- licht ---
const zon = new THREE.DirectionalLight(0xffffff, 3);
zon.position.set(5, 5, 5);
scene.add(zon, new THREE.AmbientLight(0xffffff, 0.3));

// --- jouw scene begint hier ---
const kubus = new THREE.Mesh(
  new THREE.BoxGeometry(1, 1, 1),
  new THREE.MeshStandardMaterial({ color: 0xff007d })
);
scene.add(kubus);

// --- resize ---
addEventListener('resize', () => {
  camera.aspect = innerWidth / innerHeight;
  camera.updateProjectionMatrix();
  renderer.setSize(innerWidth, innerHeight);
});

// --- de loop ---
const timer = new THREE.Timer();
renderer.setAnimationLoop(() => {
  timer.update();
  const t = timer.getElapsed();

  kubus.rotation.y = t;

  controls.update();
  renderer.render(scene, camera);
});
</script>
</body>
</html>

Werk je op de schoolcomputers of thuis naast deze cursus? Vervang dan de twee CDN-adressen door de lokale clone: "../three.js/build/three.module.js" en "../three.js/examples/jsm/" - dan werkt je sketch ook zonder internet.

Een server starten

ES-modules laden niet vanaf file:// - dezelfde beperking als loadImage() in p5. Drie manieren, kies wat bij je opstelling past:

  1. VS Code: Live Server

    Installeer de extensie "Live Server", rechtsklik op je index.html en kies "Open with Live Server". Herlaadt automatisch bij elke wijziging - het dichtst bij het p5-editor-gevoel.

  2. Python (staat vaak al op je computer)

    Open een terminal in je projectmap en typ python -m http.server. Surf naar http://localhost:8000.

  3. Een map onder een bestaande server

    Draait er al een Apache of XAMPP (zoals op de Lab44-machines)? Zet je projectmap in htdocs en surf naar localhost/jouwmap/.

Versies: waarom dat @0.184.0 in de importmap?

three.js verandert elke maand. Een versienummer vastpinnen betekent dat je sketch over een jaar nog exact zo werkt. Laat je het weg, dan krijg je "de nieuwste" - en kan een update je sketch breken terwijl jij niets veranderde.

Let op

Oudere tutorials op internet laden three.js met een gewone script-tag zonder importmap, of gebruiken THREE als globale variabele zonder import. Dat is de verouderde manier - herken je hem, weet dan dat je zulke code eerst naar het module-patroon van deze cursus vertaalt.

Weetje

Voor grotere projecten stappen ontwikkelaars over op npm en een bundler (Vite is de populairste). Dat geeft snellere herlaadtijden en nette imports, maar verandert niets aan de three.js-code zelf. Alles uit deze cursus verhuist één-op-één mee.

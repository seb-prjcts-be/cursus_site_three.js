Real 3D letters with depth, that cast shadows and shine like any other object. You need two addons for it: a FontLoader and the TextGeometry.

What you will learn
  • 3D text is just a geometry: TextGeometry + material = mesh
  • The font is a special JSON file, loaded with the FontLoader
  • Loading is asynchronous: you build the text in the callback, like with loadFont in p5
  • geometry.center() puts the pivot point in the middle of the text

The recipe

import { FontLoader } from 'three/addons/loaders/FontLoader.js';
import { TextGeometry } from 'three/addons/geometries/TextGeometry.js';

new FontLoader().load('path/to/font.typeface.json', (font) => {
  const geom = new TextGeometry('HELLO', {
    font: font,
    size: 1,        // height of the letters
    depth: 0.3,     // thickness (the third dimension!)
    curveSegments: 8
  });
  geom.center();    // otherwise the text rotates around its left side

  const text = new THREE.Mesh(geom, material);
  scene.add(text);
});

Compare with p5: loadFont in preload, then text(). Here everything happens in the load callback, because without a font there is no geometry. The rest of your sketch (renderer, loop) just keeps running while the font loads.

Note the pattern let text = null and if (text) in the loop. That is THE way to handle asynchronously loaded things - you will see it come back later with 3D models too.

Watch out

Many letters with many curveSegments become heavy geometries. For a title that is fine; for a whole paragraph of text you are better off using a plane with a canvas texture, or simply HTML on top of your canvas.

Your own fonts

The clone has a few fonts ready in three.js/examples/fonts/ (helvetiker, optimer, gentilis - each in regular and bold). If you want to use your own font, search online for a "typeface.json converter": it turns an ordinary ttf file into the JSON format that FontLoader reads.

Did you know

TextGeometry builds letters by extruding the typeface outline - the same principle as pushing a cookie cutter through dough. The depth parameter is literally how deep you push.

In p5 vroeg je "zit de muis in deze cirkel?" met wat afstandswiskunde. In 3D is dat lastiger: de muis leeft op een plat scherm, je objecten in de diepte. De Raycaster lost dat op: hij schiet een straal vanuit de camera door de muis, en vertelt welk object hij raakt.

Kernpunten
  • Muiscoördinaten zet je om naar het bereik -1 tot +1 (genormaliseerd)
  • raycaster.setFromCamera(muis, camera) richt de straal
  • intersectObjects(...) geeft de geraakte objecten, dichtstbijzijnde eerst
  • Dit heet picking - het 3D-antwoord op "zit mouseX in mijn vorm?"

Het patroon

Drie vaste stappen, die in bijna elke interactieve scene letterlijk zo terugkomen:

const raycaster = new THREE.Raycaster();
const muis = new THREE.Vector2();

addEventListener('pointermove', (event) => {
  // 1. Muis omzetten naar -1..+1 (links-onder tot rechts-boven)
  muis.x =  (event.clientX / innerWidth)  * 2 - 1;
  muis.y = -(event.clientY / innerHeight) * 2 + 1;
});

renderer.setAnimationLoop(() => {
  // 2. Straal richten vanuit de camera door de muis
  raycaster.setFromCamera(muis, camera);

  // 3. Vragen wat hij raakt
  const raak = raycaster.intersectObjects(scene.children);
  if (raak.length > 0) {
    raak[0].object.material.color.set(0xff007d);
  }

  renderer.render(scene, camera);
});

Probeer het: beweeg over de kubussen

raak[0] is altijd het dichtstbijzijnde object - de straal kan immers meerdere kubussen achter elkaar doorboren. Wil je klikken in plaats van hoveren? Gebruik een pointerdown-listener en doe de raycast daarin.

Let op

In dit voorbeeld krijgt elke kubus zijn eigen materiaal, terwijl je op de pagina Animatie & tijd leerde materialen te delen. Dat is bewust: gedeeld materiaal = gedeelde kleur. Wil je één object apart kleuren, dan heeft het een eigen materiaal nodig.

Klikken in plaats van hoveren

Maak kubussen blijvend roze zodra je erop klikt (pointerdown), en weer wit bij een tweede klik. Bewaar de toestand per kubus in userData.

Hint
addEventListener('pointerdown', () => {
  raycaster.setFromCamera(muis, camera);
  const raak = raycaster.intersectObjects(kubussen);
  if (raak.length > 0) {
    const k = raak[0].object;
    k.userData.aan = !k.userData.aan;
  }
});

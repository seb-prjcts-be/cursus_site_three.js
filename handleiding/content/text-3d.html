Echte 3D-letters met diepte, die schaduw werpen en glanzen als elk ander object. Je hebt er twee addons voor nodig: een FontLoader en de TextGeometry.

Kernpunten
  • 3D-tekst is gewoon een geometrie: TextGeometry + materiaal = mesh
  • Het font is een speciaal JSON-bestand, geladen met de FontLoader
  • Laden is asynchroon: je bouwt de tekst in de callback, zoals bij loadFont in p5
  • geometry.center() zet het draaipunt in het midden van de tekst

Het recept

import { FontLoader } from 'three/addons/loaders/FontLoader.js';
import { TextGeometry } from 'three/addons/geometries/TextGeometry.js';

new FontLoader().load('pad/naar/font.typeface.json', (font) => {
  const geom = new TextGeometry('HALLO', {
    font: font,
    size: 1,        // hoogte van de letters
    depth: 0.3,     // dikte (de derde dimensie!)
    curveSegments: 8
  });
  geom.center();    // anders draait de tekst om zijn linkerkant

  const tekst = new THREE.Mesh(geom, materiaal);
  scene.add(tekst);
});

Vergelijk met p5: loadFont in preload, daarna text(). Hier gebeurt alles in de load-callback, want zonder font geen geometrie. De rest van je sketch (renderer, loop) draait gewoon door terwijl het font laadt.

Let op het patroon let tekst = null en if (tekst) in de loop. Dat is dé manier om met asynchroon geladen dingen om te gaan - je ziet het straks ook bij 3D-modellen terugkomen.

Let op

Veel letters met veel curveSegments worden zware geometrieën. Voor een titel is dat prima; voor een hele paragraaf tekst gebruik je beter een vlak met een canvas-textuur, of gewoon HTML over je canvas heen.

Eigen fonts

In de clone zitten een paar fonts klaar in three.js/examples/fonts/ (helvetiker, optimer, gentilis - elk regular en bold). Wil je een eigen font gebruiken, zoek dan online naar een "typeface.json converter": die zet een gewoon ttf-bestand om naar het JSON-formaat dat FontLoader leest.

Weetje

TextGeometry bouwt letters door het lettertype-omtrek te extruderen - hetzelfde principe als een koekjesvorm door deeg duwen. De depth-parameter is letterlijk hoe diep je duwt.

Kinetische typografie

Maak per letter een aparte TextGeometry (lus over de tekens van een woord) en laat elke letter met zijn eigen fase golven - de 3D-versie van het klassieke p5-tekstgolf-effect.
